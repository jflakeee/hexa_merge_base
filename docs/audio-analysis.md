## 5. SFX 샘플 분석 (19~39초 구간)

### 5.1 추출 설정

```
구간: 19초 ~ 39초 (활발한 플레이 구간)
필터: highpass=f=2000, lowpass=f=8000
목적: 배경 노이즈 제거, 효과음 주파수 대역 추출
```

### 5.2 관찰된 특성

- **주파수 대역**: 2,000Hz ~ 8,000Hz에 효과음 주요 에너지 집중
- **이벤트 간격**: 약 0.5~2초 간격으로 효과음 발생
- **피치 패턴**: 머지 시 상승 피치 (낮은 숫자 → 높은 숫자)
- **지속 시간**: 개별 효과음 약 0.1~0.5초

### 5.3 주파수 분석 결과

| 주파수 대역 | 추정 음원 |
|------------|----------|
| 1,000 ~ 2,000 Hz | 타일 이동/드롭 저음 |
| 2,000 ~ 4,000 Hz | 머지 효과음 중심 주파수 |
| 4,000 ~ 6,000 Hz | 성공/달성 고음 하모닉스 |
| 6,000 ~ 8,000 Hz | 반짝임/스파클 효과 |
| 8,000 Hz+ | 고레벨 달성 특수 효과 |
